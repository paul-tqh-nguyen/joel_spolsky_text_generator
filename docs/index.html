Joel Spolsky Blog Generator

This is a Joel Spolsky Blog Generator.

The generated blogs are driven by a neural network trained on Joel Spolsky's blogs via a character-level recurrent neural network.

Character RNNs are useful for predicting the next character (good for spell correction and word completion).

However, they aren't very useful for full sentence generation (which is what this application uses it for) since they don't account for word-level semantics.

Thus, some of the sentences here won't make sense. However, character RNNs are good for generating real-sounding words (not necessarily real ones!).

If character RNNs are not the best at generating sentences, why is this application using it for that? Because it's fun to see what mistakes it makes and to see how unreasonably close it gets!
